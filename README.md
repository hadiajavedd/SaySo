SaySo - A WEB-BASED ANONYMOUS CUSTOMER FEEDBACK PLATFORM​
SaySo is a Flask-based web application developed to allow business users to create, share, and manage questionnaires, collect anonymous responses, and view insights through a simple and user-friendly interface.

Created for academic purposes by Hadia Javed (19929589)

How to Run Locally:

1. Download the zipped project and extract it.

2. Create a virtual environment:
   macOS:
   python3 -m venv venv

3. Activate the virtual environment:
   macOS:
   source venv/bin/activate

4. Install dependencies:
   pip install -r requirements.txt

5. Run the application:
   macOS:
   python3 app.py
